3. CSS Architecture

The project uses a tiered CSS system based on custom properties (Design Tokens).

  • variables.css: Root variables for colors, spacing, and transitions.
  • base.css: Global resets and shared UI elements (Loader, Nav).
  • sections.css: Section-specific styles for the homepage.
  • pages.css: Styles for inner pages and all responsive breakpoints.
